## Hardware Utilizado

* Raspberry Pi 2B
* TCS 230
* MLX 90615
* JSN - SR04
* TCRT 5000
* LCD 16x2



## Instalação

O projeto utiliza a biblioteca wiringPi, incluída por  padrão nas versões do Raspbian

```bash
$ git clone https://github.com/guilhermecv/PI2.git
$ cd PI2/Raspberry/firmware
$ make
$ ./PI2
```
